Let’s go !

首先贴上最终成果, 省得各位看官心急

  • 最普通的等式 $a = b + c$.

  • 带下标的概率关系 $P_1P_2=\bar{P_3}P_4$

  • 公式块+分段函数

初探

第一次的尝试呢, 分为了以下几步

  1. 安装hexo-math插件, 指令为npm install hexo-math --save 其中—save 是自动把模块和版本号添加到dependencies部分, 即根目录下的package.json文件中
  2. 在_config.yml中加入plugin: - hexo-math
  3. 在你的文章的*.md文件中开启开关mathjax: true
  4. 查看package.json文件, 会发现已经有了3.0.4版的的插件安装好了

按照大多数人的经验介绍, 和老版本的插件特性, 写到这里就已经算是成功了, 但是无数小白都在这里折戟沉沙, 我们可以以2017-04-19为日期分割线, 因为原作者在这里进行了一次更新, 插件进入到了3.0.0的时代, 具体发生了神马变化可以自行参考上面的传送门

回溯

上回书说道, 新版本的插件对于我们不奏效, 为了解决这个问题, 正常人都会顺理成章的产生两个思路

  • 探索新版的下的使用方法
  • 安装旧版本, 因为教程和经验多啊, 广大学习python的小白有多少是因为教程和例程只适用于2.7和放弃了研究3.x啊hhhhhh

如果想继续使用老版本可以如下

  1. 卸载一波刚刚安装的插件, npm uninstall hexo-math --save
  2. 按照版本号安装对应的hexo-math插件, 例如笔者选择了1.0.8版本, 指令即为npm install hexo-math@^1.0.8 --save

笔者亲测有效
